# 🚀 TransApp - Быстрый старт

## 📱 Разработка на физическом Android устройстве

### 1️⃣ Подключите телефон

```bash
# Проверьте подключение
npm run device:list
```

### 2️⃣ Запустите приложение

```bash
# Автоматическая настройка + запуск
npm run android:device

# Или для конкретного устройства (если подключено несколько)
npm run android:device:specific
```

### 3️⃣ Готово! 🎉

Приложение установится и запустится на вашем телефоне.

---

## 🛠️ Полезные команды

| Команда | Описание |
|---------|----------|
| `npm run android:device` | Настроить порты + запустить на устройстве |
| `npm run android:device:specific` | Запустить на конкретном устройстве (28J9K24319016011) |
| `npm run setup:device` | Только настроить порты (без запуска) |
| `npm run device:list` | Показать список подключенных устройств |
| `npm run device:logs` | Показать логи приложения |
| `npm run device:clear` | Очистить данные приложения |

## 🐛 Решение проблем

### Белый экран на телефоне

```bash
# 1. Настройте порты
npm run setup:device

# 2. Перезапустите приложение на телефоне
```

### Ошибка "more than one device"

```bash
# Используйте команду для конкретного устройства
npm run android:device:specific
```

### Приложение крашится

```bash
# Очистите кэш и переустановите
npm run device:clear
npm run android:device
```

## 💡 Советы

1. **Держите Metro bundler запущенным:**
   ```bash
   npm start
   ```

2. **Dev Menu на телефоне:**
   - Встряхните телефон
   - Или: `adb shell input keyevent KEYCODE_MENU`

3. **Быстрая перезагрузка:**
   - Dev Menu → Reload
   - Или дважды нажмите `R` в терминале Metro

4. **Просмотр логов в реальном времени:**
   ```bash
   npm run device:logs
   ```

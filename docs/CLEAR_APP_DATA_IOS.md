# 🗑️ Очистка данных приложения на iOS

## 🎯 Когда это нужно?

- Приложение показывает экран PIN вместо экрана авторизации
- Нужно протестировать первый запуск приложения
- Нужно сбросить все настройки и данные

---

## 📱 Способы очистки данных

### Способ 1: Через npm скрипт (рекомендуется)

```bash
# Удалить приложение и переустановить
npm run ios:clear

# Или полностью сбросить симулятор (удалит ВСЕ данные)
npm run ios:fresh
```

---

### Способ 2: Вручную через командную строку

```bash
# Удалить приложение с активного симулятора
xcrun simctl uninstall booted org.reactjs.native.example.Transappru

# Переустановить
npm run ios
```

---

### Способ 3: Через кнопку в приложении (DEV режим)

1. Запустите приложение в dev режиме
2. Дождитесь экрана загрузки (spinner)
3. Нажмите красную кнопку **"🗑️ Очистить данные (DEV)"** внизу экрана
4. Приложение покажет экран авторизации

**Примечание:** Кнопка видна только в dev режиме (`__DEV__ = true`)

---

### Способ 4: Удалить через Simulator

1. Откройте iOS Simulator
2. Найдите иконку приложения **TransApp**
3. Долгое нажатие на иконку
4. Нажмите **"Remove App"** → **"Delete App"**
5. Переустановите: `npm run ios`

---

### Способ 5: Сбросить весь симулятор

```bash
# Узнать список симуляторов
xcrun simctl list devices

# Стереть конкретный симулятор
xcrun simctl erase <DEVICE_ID>

# Или стереть все симуляторы
xcrun simctl erase all
```

**⚠️ Внимание:** Это удалит ВСЕ данные со всех симуляторов!

---

## 🔍 Проверка результата

После очистки данных приложение должно:

1. ✅ Показать экран **ввода номера телефона** (AuthScreen)
2. ✅ Не показывать экран PIN
3. ✅ Не иметь сохранённого токена

## 📝 Сравнение с Android

| Действие | Android | iOS |
|----------|---------|-----|
| Очистить данные | `npm run android:clear` | `npm run ios:clear` |
| Полный сброс | `npm run android:fresh` | `npm run ios:fresh` |
| Кнопка в приложении | ✅ Работает | ✅ Работает |

## 🐛 Troubleshooting

### Ошибка: "No devices are booted"

```bash
# Запустите симулятор сначала
open -a Simulator

# Затем выполните команду
npm run ios:clear
```

### Ошибка: "Unable to find a destination"

Укажите конкретный симулятор:

```bash
xcrun simctl list devices | grep Booted
xcrun simctl uninstall <DEVICE_ID> org.reactjs.native.example.Transappru
```

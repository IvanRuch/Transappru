# Legacy Documentation Archive

Архив миграционной эпохи проекта (≈ Nov 2025 – Apr 2026). 57 markdown-файлов:
completion-логи, fix-логи, setup-гайды того периода. Содержат исторический
контекст, который иногда полезен для git-археологии (почему такая структура,
откуда то или иное архитектурное решение).

## Read-only

Эти файлы **не редактируются и не обновляются**. Они отражают состояние
проекта на момент написания, не текущую реальность. SDK-версии, пути и
рекомендации внутри — устаревшие.

## Где живёт актуальная документация

- **Проектная документация (single source of truth):** `Writerside/topics/`
  → `Writerside/topics/landing.md` для входа.
- **Vendor docs (Kazna API spec/contract):** `payment-service/docs/vendor/kazna/`.

См. `Writerside/topics/decision-log.md` → ADR-013.

## Если нужно вытащить что-то отсюда

Не редактируйте файл здесь и не ссылайтесь на него извне архива.
Перенесите релевантную часть в соответствующий topic в `Writerside/topics/`,
обновите под нынешнее состояние кода, и закоммитьте — это закроет «миграцию»
для конкретного куска контента.

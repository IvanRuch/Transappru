1. Откройте `START_HERE.md` в корне
2. Следуйте инструкциям
3. При необходимости открывайте файлы из `docs/`

### Поиск документации
1. Откройте `docs/README.md`
2. Найдите нужный раздел
3. Откройте соответствующий файл

### Добавление новой документации
```bash

# Добавьте ссылку в docs/README.md
```

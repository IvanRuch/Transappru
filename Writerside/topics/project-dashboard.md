- **2026-04-22**: Upgrade `WebSidebar.loadData` from `inFlightRef` dedup to `AbortController` with **latest-wins** semantics. Reason: dedup drops new triggers while an old request hangs вАФ meaning a stalled backend (ivan.trans-konsalt.ru sometimes takes >30s вЖТ `AxiosError ECONNABORTED`) blocks the user for the full axios timeout before they can retry.
